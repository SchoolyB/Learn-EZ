# Supported Data Types in EZ

###Primitives

- byte   //0 to 255
- char ''
- string ""
  - Raw string ``
- int   //Negative or positive Whole numbers...Your CPU sets the limit
- uint  //Positive whole numbers...Your CPU sets the limit
- float //negative or positive decimal numbers 
- bool  //true or false

### SIZED INTEGERS

- i8   //-128 to 127
- i16  //-32,768 to 32,767
- i32  //-2.1 billion to 2.1 billion
- i64  //-9.2 quintillion to 9.2 quintillion
- i128 //CRAZY RANGE
- i256 //DO YOU EVEN NEED THIS???

- u8   //0 to 255
- u16  //0 to 65,535
- u32  //0 to 4.2 billion
- u64  //0 to 18.4 quintillion
- u128 //CRAZY RANGE
- u256 //DO YOU EVEN NEED THIS???

### Collections

- Arrays(Dynamic and Fixed)
- Maps(Key value pairs) 

### User Defined Types
- Types created by the user using structs or enums
